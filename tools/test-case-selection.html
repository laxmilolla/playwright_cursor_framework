1 DS1 Navigate to Data Submissions and verify page User is logged in to hub (or will log in via Login.gov). 1. Go to hub. 2. Log in (Login.gov: email, password, OTP, Grant). 3. Click "Data Submissions". 4. Wait for /data-submissions. Page shows "Data Submissions" heading, "Create a Data Submission" button, and "Submission Name" column.
2 DS11 Create a Data Submission (CTDC, study, name) On Data Submissions page. 1. Click "Create a Data Submission". 2. Open Data Commons, select CTDC, click Select. 3. Select a study. 4. Fill Submission Name (e.g. Laxmi_Playwright). 5. Click Create. Dialog closes; Data Submissions list is visible; new submission appears or list refreshes.
3 DS3 Filter by Program (Organization) On Data Submissions page. 1. Open Program/Organization dropdown (All). 2. Select an organization by visible text. Table shows only rows matching the selected program.
4 DS2 Filter by Submission Name On Data Submissions page. 1. Type value in Submission Name filter (e.g. "upload"). 2. Press Enter or wait for filter. Table shows only rows where Submission Name contains the entered value.
5 DS4 Filter by Status (e.g. New, Completed) On Data Submissions page. 1. Click "statuses selected" button. 2. Select one or more statuses (e.g. New, Completed). Table shows only rows matching the selected status(es).
6 DS7 Filter by dbGaP ID On Data Submissions page. 1. Type value in dbGaP ID filter (min 3 chars, e.g. "23" or "phs"). 2. Apply filter. Table shows only rows where dbGaP ID matches (or no rows if no match).
7 DS5 Filter by Data Commons (e.g. ICDC) On Data Submissions page. 1. Open Data Commons dropdown (All). 2. Select option by visible text (e.g. ICDC). Table shows only rows matching the selected Data Commons.
8 DS6 Filter by Submitter On Data Submissions page. 1. Open Submitter dropdown (All). 2. Select submitter by visible text (e.g. "laxmi Lolla"). Table shows only rows matching the selected submitter.
